Bootstrap 4 - Introduction

Cette page fonctionne avec bootstrap 4 en CDN

Présentation générale

  • Bootstrap est un framework CSS "cadriciel"
  • c'est une librairie côté client (front-end)
  • Il nous aide à mettre en forme les pages web.
  • C'est une librairie CSS mais il y a également des composants HTML et JavaScript
  • Basé sur Sass(css avancé) et normalize
  • Il comporte également une grille de mise en page

Intérêt

  • Uniformisation pour les navigateur
  • Gain de temps
  • Styles identiques
  • La grille responsive !!!
  • PLeins de suppléments en +
  • Gratuit
  • Bonne gestion des médias queries
